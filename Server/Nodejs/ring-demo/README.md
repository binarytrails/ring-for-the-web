# Ring-demo

## Installation

Node dependecies (locally) from package.json:

    npm install

Node-gyp (globally):

    sudo npm install -g node-gyp

It [depends](https://www.npmjs.com/package/node-gyp) on Python, Make, and a C++ compiler like GCC.

## Compiling

    ./build.sh

## Run

    ./server.js

## Help

    ./server.js -h
